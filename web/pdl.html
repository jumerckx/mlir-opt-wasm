mlir-opt · PDL → pdl_interp lowering

PDL rewrite patterns are lowered to the executable pdl_interp dialect before being run by the bytecode interpreter. There are two ways to get there. The direct path runs convert-pdl-to-pdl-interp in one shot. The match-dialect path goes through the match dialect, where independent matchers can be combined into a single shared decision tree (switch_op_name, common-prefix sharing) before being lowered. Edit the patterns below and press Run to see every intermediate step. Click an op name to link it to its lowered descendants across panes.
